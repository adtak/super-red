## Project Overview

This is an Expo/React Native project (super-red).

## Development Commands

```bash
# Install dependencies
npm install

# Start Expo development server
npx expo start

# Run on iOS simulator
npx expo run:ios

# Run on Android emulator
npx expo run:android
```

## Environment

- `EXPO_DEBUG=1` is set for debugging
